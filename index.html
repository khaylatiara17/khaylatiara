Proyek Saya

Proyek 1: Web Development

Membuat situs web responsif dengan desain modern dan fokus pada pengalaman pengguna (UX). Saya berfokus pada pengembangan front-end menggunakan HTML, CSS, dan JavaScript, serta integrasi dengan backend API untuk fungsi dinamis.

Proyek 2: Aplikasi Mobile

Mengembangkan aplikasi lintas platform untuk melacak tujuan kebugaran pengguna. Aplikasi ini memungkinkan pengguna untuk mengatur dan melacak latihan harian, memantau kemajuan kebugaran mereka, dan mendapatkan saran personalisasi berdasarkan data yang dimasukkan.

Proyek 3: Sistem Manajemen Inventaris

Membangun aplikasi berbasis web untuk memanajemen inventaris produk. Sistem ini menggunakan database untuk menyimpan data produk dan memungkinkan pengguna untuk melakukan pencarian, menambah, mengedit, dan menghapus data produk dengan mudah.
